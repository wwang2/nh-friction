## Refined Research Question

Find the odd continuous friction function g(ξ) for the single-auxiliary-variable Nosé-Hoover thermostat that minimizes the difficulty-weighted mean integrated autocorrelation time τ_int of the position observable across a benchmark suite of 3 canonical target distributions (1D harmonic oscillator, 2D double-well, 2D Gaussian mixture), while preserving the canonical Boltzmann distribution to within KL < 0.05.

## Metric

- **Name:** Difficulty-weighted mean integrated autocorrelation time (weighted_tau_int)
- **Formula:**
  ```
  weighted_tau_int = Σ_k  w_k · (1/S) Σ_s τ_int(k, s)

  where:
    k ∈ {harmonic_1d, doublewell_2d, gaussmix_2d}     (benchmark potentials)
    s ∈ {42, 137, 2024}                                 (seeds)
    S = 3                                               (number of seeds)
    w_k = τ_int^{NHC}(k) / Σ_j τ_int^{NHC}(j)        (difficulty weights, measured at baseline)

  τ_int(k, s) = 1 + 2 · Σ_{t=1}^{W} C(t)/C(0)
    C(t) = (1/N) Σ_{i=1}^{N-t} (q_i - q̄)(q_{i+t} - q̄)
    W = min(W* | C(W*)/C(0) < 0.05 or W* > 6·τ_int_running_estimate)
  ```
  (Sokal automatic windowing, cutoff 6×τ_int; Sokal 1997 eq. 3.3)

- **Direction:** minimize
- **Provenance:** Sokal, A.D. (1997). "Monte Carlo Methods in Statistical Mechanics: Foundations and New Algorithms." Lecture notes, Cargese Summer School. Eq. 3.3. Also: Chodera, J.D. (2016). J. Chem. Theory Comput. 12, 1799. doi:10.1021/acs.jctc.5b00784
- **Target:** weighted_tau_int < 150 (≥30% improvement over NHC M=3 baseline, expected ~200–500 unweighted)
- **Known pitfalls:**
  - *Ergodicity failure (infinite τ_int):* Standard NH g(ξ)=ξ gets trapped in KAM tori on 1D HO → τ_int → ∞. Guard: cap τ_int at N_steps/2 = 500,000 to keep metric finite; report as "non-ergodic" flag.
  - *Canonical distribution gaming:* Solution could return a trajectory that looks canonical by construction. Guard: hard KL < 0.05 gate (disqualify before computing τ_int).
  - *Oddness violation:* g(ξ) ≠ −g(−ξ) breaks Liouville. Guard: numerical oddness check |g(ξ) + g(−ξ)| < 1e-6 at 100 test points; disqualify if violated.
  - *Seed exploitation:* Online optimizer might reverse-engineer test data from seed. Guard: seeds are fixed but simple integers, not cryptographic; KL gate catches memorization.

## Evaluation Procedure

1. **Validate solution interface:**
   - Import `solution.py`; check `friction_function` and `friction_derivative` are callable
   - Oddness check: max |g(ξ) + g(−ξ)| < 1e-6 on 100 log-spaced ξ ∈ [0.01, 10.0] → disqualify if violated
   - Derivative check: max |g'(ξ) − (g(ξ+h) − g(ξ−h))/(2h)| / (|g'(ξ)| + 1) < 1e-4 for h=1e-5 → warn if violated

2. **For each benchmark potential k ∈ {harmonic_1d, doublewell_2d, gaussmix_2d}:**
   - For each seed s ∈ [42, 137, 2024]:
     - Initialize: q ~ N(0,1), p ~ N(0,1), ξ = 0.0 (state seeded with `np.random.default_rng(s)`)
     - If `setup(s)` defined: call it now
     - Integrate Nosé-Hoover ODEs for 10,010,000 steps, dt=0.01 (discard first 10,000 as burn-in):
       ```
       # Each full step:
       p += -∇V(q) · dt/2          # half kick
       p *= exp(-g(ξ) · dt/(2Q))   # exact exponential friction half-step
       q += p/m · dt                # drift
       p *= exp(-g(ξ) · dt/(2Q))   # exact exponential friction half-step
       p += -∇V(q) · dt/2          # half kick
       ξ += (p²/m - d·kT)/Q · dt   # thermostat update (d = system dimension)
       ```
       Parameters: m=1, kT=1, Q=1; record q[0] every 10 steps → N=100,000 samples
     - Compute KL divergence: bin q[0] into 100 equal bins on [−6, 6]; compare to analytical marginal of exp(−βV); if mean KL across seeds > 0.05 for any potential → disqualify, METRIC=inf
     - Compute τ_int via Sokal windowing on the 100,000 samples; cap at 50,000 if non-convergent

3. **Compute difficulty weights** (frozen at baseline eval time, stored in `research/eval/config.yaml`):
   ```
   w_k = tau_int_NHC(k) / sum_j tau_int_NHC(j)
   ```

4. **Compute final metric:**
   ```
   weighted_tau_int = sum_k w_k · mean_s(tau_int(k, s))
   ```

5. **Report auxiliary metrics** (logged, not used for ranking):
   - `ESS_per_second` = N / (tau_int × wall_time_s) per potential
   - `ergodicity_score_1d_HO` = KS p-value (p > 0.05 = ergodic) + variance_ratio (var_sample / var_Boltzmann, ideal = 1.0)
   - `kl_per_potential`: KL divergence per potential
   - `tau_int_per_potential`: unweighted τ_int per potential and seed

6. `print(f"METRIC={weighted_tau_int:.6f}")`

## Benchmark Potentials

| Name | Dimensionality | Formula | Analytical marginal |
|------|---------------|---------|---------------------|
| `harmonic_1d` | d=1 | V(q) = q²/2 | N(0,1) |
| `doublewell_2d` | d=2 | V(q,y) = (q²-1)² + y²/2 | bimodal on x, N(0,1) on y |
| `gaussmix_2d` | d=2 | V = -log Σ_{k=0}^{4} exp(-\|q - μ_k\|²/2), μ_k = 3·(cos(2πk/5), sin(2πk/5)) | 5-mode GMM, σ=1, uniform weights |
